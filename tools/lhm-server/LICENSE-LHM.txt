--- What this means for lil_bro ---

lhm-server.exe is a separate binary that references LibreHardwareMonitorLib
via NuGet.  Under MPL-2.0, the library source remains open and available at
the GitHub link above.  lil_bro itself is not subject to MPL-2.0 copyleft
requirements because it communicates with lhm-server.exe only over HTTP.
